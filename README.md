### 相关说明及注意点请查看[博客说明](https://ygkkk.blogspot.com/2022/11/hysteria-yg-youtube.html)

```
wget -N https://gitlab.com/rwkgyg/hysteria-yg/raw/main/hysteria.sh && bash hysteria.sh
```
